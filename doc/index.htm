PilRC is an application that takes a resource script file and generates one or more binary resource files that are to be used when developing for the Palm Computing Platform. PilRCUI gives you a preview of your resource file.

*NOTE* color support available NOW *NOTE*

The source code and binaries are available free of charge and are Copyrighted 1997-2000 by Wes Cherry and Aaron Ardiri. Any modifications (bug-fixes and improvements) to PilRC should be sent to Aaron Ardiri so that they can be integrated into the next release; so the entire Palm Computing Platform development community can obtain the latest and greatest from PilRC.

PilRC is a platform independant program. It should compile under Win32, Unix, OS/2 and MSDOS using your favourite C/C++ compiler. Win32 binaries are provided and compiled using Cynus Cygwin C/C++ tools. You can download the C/C++ compiler suite for Win32 at http://www.cygnus.com
